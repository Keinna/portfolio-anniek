About me

I am an Alkmaar / Amsterdam based Front End Developer. Ever since I discovered coding, Visual Studio Code is my new best friend. I find it fascinating to see how you can turn your ideas into real projects. Because I love to learn new things, becoming a Front End Developer is the best step I ever took (even though sometimes I want to throw my computer out of the window). So far I have learned HTML5, CSS3, SASS, Bootstrap 4, Javascript, Node / NPM , Webpack and took a peak at some Backend stuff. At the moment I am focusing on the React framework and the Harvard CS50 course. Please check out my portfolio or my Github account if you want to see what I have done already.
