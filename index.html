I like to create websites that look attractive and have great user experience at the same time. I love how you can turn a design into something real. Sometimes I can get lost in the details because I just want to add a little something extra.

Currently I am working at a digital agency in Alkmaar, the Netherlands. The past years I have been building websites/features for different clients with all kinds of requests. The projects range from small to really large. To accomplish this I have worked with several other developers/designers/projectmanagers in a scrumteam where PHP is being used for the backend.

Every day I try to write better code. In a way that it is more reusable and easy to maintain. I keep in mind that other people have to read and understand my code as well.

One of my next goals is to get a really good grasp of REACT.

Javascript e-commerce project

Highlights project

  • Total number of items in cart visible in navigation
  • Checkout cart functionality
  • Loads data from 'database'
  • Vanilla Javascript project

Total number of items in cart visible in navigation
This badge will always be visible in the navigation. It shows the total amount of items in your cart. Thanks to local storage it will always show the right amount of items. One click brings you to the cart with the right amount of items and the right price

Checkout cart functionality
You can add or remove items. It is also possible to clear the whole cart at once. At the bottom of the cart you can see the total amount before you purchase.

Loads data from 'database'
The data for the items is coming from a different file. This file mimicks a database. When you add data to the file, the javascript will display the right price, name, image and description in the store dynamically.

Vanilla Javascript project
However it would probably be easier to use a framework to set up a store. This project was a great way to practice my vanilla javascript skills.
