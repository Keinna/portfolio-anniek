About me

I like to create websites with distinct features and have great user experiences. I find it fascinating how you can turn your ideas into real projects. Currently, I am working at a digital marketing agency, Reyez, Alkmaar, the Netherlands. The past years I am building websites for different clients with all kinds of framework. Amongst other, HTML5, CSS, SASS, Bootstrap, Node, NPM, Javascript, React, Harvard, CS50 and Figma design github git bitbugged at lassian. One of my main focuses is to write solid code which is easy accessible. In my opinion solid code is more efficient and helps to make a website more user friendly. I also like to build websites for private individuals, to help them to build ambitious but accessible web projects.
